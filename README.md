# 소금빵 포스기 (React)

8개 메뉴를 클릭해 수량을 올리고, **총 주문 빵 개수**와 **합산 가격**을 크게 보여주는 간단한 포스기 화면입니다.

## 실행 방법

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속.

## Vercel 배포

이 프로젝트는 정적 파일이라 Vercel에 바로 배포할 수 있습니다.

1. GitHub 저장소 연결
2. Framework Preset: `Other`
3. Build Command: 비워두기
4. Output Directory: `.`
